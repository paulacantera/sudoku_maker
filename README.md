# Sudoku Maker

A React application for generating and solving Sudoku puzzles.

## Features

- **Generate puzzles** with three difficulty levels: Easy, Medium, and Hard
- **Interactive board** – click a cell and use the number pad (or keyboard) to fill it in
- **Validation** – incorrect entries are highlighted in red
- **Win detection** – a congratulations banner appears when the puzzle is solved
- **Keyboard navigation** – arrow keys to move, 1-9 to enter digits, Backspace/Delete to erase

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

|---------|-------------|
